Domino Pro - Gestor de Torneos

Aplicación web full-stack diseñada para la gestión integral de torneos de dominó, automatizando desde la inscripción hasta la clasificación final. La plataforma implementa lógicas de negocio complejas basadas en reglamentos oficiales para garantizar la transparencia y competitividad del evento.

Funcionalidades Clave:

  • Gestión de Torneos: Creación de eventos con parámetros personalizables (rondas, puntos, tiempo) y generación de un código único para seguimiento público.
  • Sistema de Emparejamiento Suizo: Generación automática de rondas, emparejando jugadores según su rendimiento para mantener la competitividad.
  • Ranking en Tiempo Real: Actualización instantánea de la clasificación después de cada partida registrada.
  • Módulo Avanzado de Sanciones: Implementación de un sistema de tarjetas (amarilla, roja, negra) con lógicas de penalización.

Tecnologías Utilizadas:

React HTML5 / CSS3 JavaScript (Vanilla JS) Firebase Bootstrap Serverless

Nexus Play - Plataforma de Contenido Corporativo

Aplicación web profesional para la gestión de contenido multimedia en múltiples pantallas de forma remota. NexusPlay permite a los usuarios controlar playlists visuales y musicales, registrar dispositivos de forma segura y actualizar el contenido en tiempo real sin necesidad de intervención manual.

Funcionalidades Clave:

  • Gestión de Contenido: Ciclo completo de subida, visualización y eliminación de imágenes, videos y audio.
  • Asignación Dual de Playlists: Asignación independiente de playlists visuales y de audio a cada pantalla.
  • Emparejamiento Seguro de Pantallas: Registro de dispositivos mediante códigos únicos de un solo uso.
  • Programación Avanzada de Contenido: Asignación de playlists a franjas horarias y días de la semana.
  • Reproducción Inteligente: El reproductor silencia y reanuda la música de fondo automáticamente.
  • Actualizaciones en Tiempo Real: El contenido se actualiza automáticamente sin recargar la página.
  • Autenticación Profesional: Registro seguro con verificación de correo e inicio de sesión con Google.
  • Plataforma White-Label y Gestión de Roles: Arquitectura preparada para marca blanca y perfiles de usuario.

Tecnologías Utilizadas:

React Firebase Serverless
